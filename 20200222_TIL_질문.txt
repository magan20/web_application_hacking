1. 서버가 클라이언트를 통한 데이터 전송을 하는 이유
2. Set-Cookie 키 값에는 세션 아이디와 사용자 식별자 등이 들어있다.
3. asp.net: 웹 개발자가 동적 웹 사이트, 응용 프로그램 및 서비스를 구축할 수 있도록 설계된 오픈 소스 프레임워크
4. viewstate: 클라이언트 측 상태 관리 기술, 웹 페이지를 게시 할 때 페이지에 사용자 데이터를 저장하는데 사용
5. etag: 캐시 가능한 리소스에 서버가 부여하는 일종의 일련번호를 저장, 해당 리소스가 업데이트 될때마다 변경
6. if-modified-since: 지정된 날짜가 저장

8. 애플릿: 플러그인의 하나로서 전용 윚ㅅ 엔진이나 더 큰 프로그램 범위 내에서 실행되는 특정한 작업을 수행하는 작은 응용 프로그램
- 바이트코드: 가상 컴퓨터에서 돌아가는 실행 프로그램을 위한 이진 표현법
중급 언어임
- 체크섬: 중복 검사의 한 형태, 오류 정정을 통해 자원의 무결성을 보호하는 방법
- jar(자바 아카이브): 여러개의 자바 클래스 파일과, 클래스들이 이용하는 관련 리소스 및 메타데이터를 하나의 파일로 모아놓은 패키지 파일 포맷
- 콜 스택: 컴퓨터 프로그램에서 현재 실행중인 서브루틴에 관한 정보를 저장하는 스택 자료구조이다.
- 서브루틴: 소프트웨어에서 특정 동작을 수행하는 일정 코드 부분 
- 액티브x 컨트롤: 액티브x 기술을 사용하는 컨트롤, 애니메이션을 허용하여 탐색을 더 쉽게 하거나 보안 업데이트와 같은 설치 작업을 도와줌
- 어도비 플래쉬: 응용 프로그램을 제작하는데 사용되는 멀티미디어 및 소프트웨어 플랫폼
- SWF: 소크웨이브 플래쉬 파일 형식의 파일 확장자, small wev format을 나타냄, 비디오 및 벡터 기반 애니메이션과 사운드를 포함, 웹을 통해 효율적으로 전달하도록 설계되기도
